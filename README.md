# 📊 E-Commerce Sales Analytics Dashboard

This project focuses on analyzing e-commerce sales data and building an interactive dashboard using Power BI to derive meaningful business insights.

---

## 🚀 Project Overview

The objective of this project is to:

* Analyze sales performance across categories, regions, and time
* Identify top-performing products and regions
* Visualize trends using an interactive dashboard

---

## 🛠️ Tools & Technologies

* **Microsoft Excel** – Data Cleaning & Preprocessing
* **Power BI** – Data Visualization & Dashboard Creation

---

## 📂 Dataset

* Sample Superstore Dataset (CSV/Excel format)
* Includes:

  * Order Date
  * Sales
  * Profit
  * Category
  * Region

---

## 📊 Dashboard Features

* 📌 **KPIs**

  * Total Sales
  * Total Orders
  * Total Profit

* 📈 **Visualizations**

  * Sales by Category (Bar Chart)
  * Sales by Region (Pie Chart)
  * Monthly Sales Trend (Line Chart)

* 🎛️ **Filters**

  * Region-based slicer for dynamic analysis

---

## 🔍 Key Insights

* Technology category generated the highest sales
* West region contributed the most revenue
* Sales peaked during the beginning of the year
* Interactive filters allow real-time data exploration

---

## 📁 Project Files

* `cleaned_data.xlsx` – Processed dataset
* `Ecommerce_Sales_Dashboard.pbix` – Power BI dashboard file

---

## 💡 Conclusion

This project demonstrates the use of data analytics and visualization tools to extract actionable insights and support business decision-making.
